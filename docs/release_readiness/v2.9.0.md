# Release readiness — v2.9.0

This document is internal release telemetry. It is NOT shown on the
public landing page. The release rule below is enforced by tests and by
`visionservex readiness verdict`.

> **Release rule.** A row is release-ready when
> `functional_readiness >= 90` OR
> (`operational_readiness >= 90` AND `blocker_certainty >= 95`).
> `visionservex readiness verdict --json` returns `RELEASE_OK` when every
> row passes; otherwise `RELEASE_BLOCKED`.
