## v3 release justification

v3.0.0 is the first major version of VisionServeX because:

1. All public CLI commands work and return structured results.
2. The model load matrix covers all 113 registry models with zero core failures.
3. The readiness verdict is RELEASE_OK for all 20 factors.
4. Optional extras have been smoke-tested on GitHub Actions (anomaly, tracking, ReID, OpenMMLab).
5. Sidecar Dockerfiles are pinned, documented, and have a GHA publish workflow.
6. The clean-wheel install passes all 7 gates from a fresh venv.
7. CI tests are green (MedSAM fast-CI bug fixed).

The only pending action is Docker image publication to GHCR, which happens automatically when the release is published via the `publish-sidecars.yml` workflow.
